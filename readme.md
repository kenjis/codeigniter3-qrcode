## Installation

### Setup Database

```
$ mysql -uroot
```

```
mysql> CREATE DATABASE `codeigniter3_qrcode` DEFAULT CHARACTER SET utf8mb4;
```

```
$ mysql -uroot codeigniter3_qrcode < 'database/codeigniter3_qrcode.sql'
```

### Run built-in Web Server

```
$ bin/server.sh
```
